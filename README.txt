3D Print Online Calculator

Proyecto final realizado para el curso de Javascript de Coderhouse.
Se buscó crear un cotizador online de Impresión 3D, donde el usuario pudiera ingresar los datos de su modelo, elegir el tipo de material, crear y comparar presupuestos.
Se trabajaron conceptos como manejo de DOM, eventos, fetch, JSON, storage, uso de librerías, etc.

Final project for Coderhouse´s Javascript course.
The aim was to create an online 3D Printing calculator, where the user could enter the data of their 3D model, choose the type of material, create and compare budgets.
Concepts such as DOM management, events, fetch, JSON, storage, use of libraries, etc. were worked on.
